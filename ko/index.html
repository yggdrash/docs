# Getting Started

Install YGGDRASH Commander:

yarn global add @yggdrash/cli # or npm install -g @yggdrash/cli

Build Your BranchChain:

ygg branch init
ygg branch build
ygg branch deploy

Running Node:

docker run --rm -p 8080:8080 -v $HOME/.yggdrash:/.yggdrash yggdrash/yggdrash-node
